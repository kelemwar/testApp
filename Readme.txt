 Architecture my App
    - class Actions
        contains enums action keys
    - class App
        contains main class for run application
    - class Contact

        also contains getter and setter for it
    - class DAO
        contains methods for managing phonebook data
     - class DAO

I chose the maven project because when building the project into an executable file you can specify all the necessary dependencies.

To run the application from the command line, you must download jar-file and run the command "java -jar +path for jar file"
